conda安装mysqlclient遇到的问题

canda安装mysqlclient遇到的问题

我的PC有三个python环境，所以有时安装模块总会遇到一些问题

C:\Users\Mto\Desktop>py -2 -V
Python 2.7.15

C:\Users\Mto\Desktop>py -3 -V
Python 3.8.3

此外还有conda的环境

C:\Users\Mto>conda -V
conda 4.8.4

原本想着conda install mysqlclient一句话的事

看下图，提示

conda提示不兼容，但我用py3之前安装成功了

报错

UnsatisfiableError: The following specifications were found
to be incompatible with the existing python installation in your environment:

Specifications:

  - mysqlclient -> python[version='>=2.7,<2.8.0a0|>=3.6,<3.7.0a0|>=3.7,<3.8.0a0|>=3.5,<3.6.0a0']

Your python: python=3.8

If python is on the left-most side of the chain, that's the version you've asked for.
When python appears to the right, that indicates that the thing on the left is somehow
not available for the python version you are constrained to. Note that conda will not
change your python version to a different minor version unless you explicitly specify
that.

The following specifications were found to be incompatible with your CUDA driver:

  - feature:/win-64::__cuda==10.2=0

Your installed CUDA driver is: 10.2

解决方法

下载库，使用conda里的pip进行安装，上面虽然提示兼容问题，但我测试可以使用

下载mysqlclient

https://pypi.org/project/mysqlclient/#files

选择一个版本

使用conda当前环境的pip进行安装

使用conda env list查看当前使用的环境

找到环境下的pip在下载mysqlclient路径下执行以下命令

conda环境路径\Scripts\pip.exe install 库文件
